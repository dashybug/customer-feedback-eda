# customer-feedback-eda
A comprehensive Exploratory Data Analysis (EDA) of banking customer feedback and transaction datasets. This project utilizes Python (Pandas, Seaborn) and Dimensionality Reduction (PCA) to identify customer segments, visualize satisfaction trends, and map feedback to specific financial product offerings.

Data Integration: Merged transaction, feedback, and product data using sequential mapping based on Customer IDs.

Imputation: Handled missing values in satisfaction scores and transaction amounts using median imputation.

Dimensionality Reduction: Implemented Principal Component Analysis (PCA) to visualize customer engagement and satisfaction variance.

Visualizations: Created multi-dimensional scatter plots and heatmaps to uncover relationships between product types and risk levels.
